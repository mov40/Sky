1. Ошибка обработки None в capitalize
Метод capitalize вызывает ошибку AttributeError при передаче None.  
Ожидаемый результат: Генерировать исключение TypeError с сообщением "Argument cannot be None".

2. Некорректная обработка строки из пробелов в trim
Метод trim очищает строку из пробелов полностью, хотя должен оставить её без изменений.  
Ожидаемый результат: Оставлять строку с пробелами без изменений.

3. Ошибка обработки пустой строки в contains
Метод contains возвращает True, если передан пустой символ "".  
Ожидаемый результат: Возвращать False, если символ пустой.

4. Неправильное удаление русских символов в delete_symbol
Метод delete_symbol некорректно удаляет русские символы из строки.  
Ожидаемый результат: Полностью удалять указанный символ из строки.

5. Ошибка обработки None в delete_symbol
Метод delete_symbol вызывает ошибку AttributeError при передаче None.  
Ожидаемый результат: Генерировать исключение TypeError с сообщением "Arguments cannot be None".
